Class ScProject

The project.

Description

This is a sub-singleton of class Scribe, and is initialized on it's constructor. It may be accessed through the main singleton (the $Sc global variable) using something like the example below:

 global $Sc;
 echo "The project's name is:" . $Sc->Project->getName();
