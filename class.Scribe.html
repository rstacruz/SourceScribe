Class Scribe

The main singleton and front controller.

Description

This is instanciated as the global variable $Sc. This class is responsible for reading the configuration file. It also holds the ScProject sub-singleton.

Properties

$Project
The ScProject sub-singleton.
$Readers
read-only Key/value pairs of file reader drivers.
$_config
read-only Raw data from the scribe.conf file (after being YAML-parsed).
