Class ScBlock

A page of sorts.

Every comment block is defined as an ScBlock. A reader (ScReader) will scan files, and for every comment block it encounters, it uses ScProject::register() to register a block instance to the project, which is created through factory().

Each block has:

  • A block type
  • Title
  • Brief description (optional)
  • Content
  • Members

Subclassing

This class provides a few protected methods. These methods are provided to be overridden for subclasses.

Factory

factory()
static Creates an ScBlock instance with the right class as needed.

Protected methods

ScBlock()
protected The constructor. (Don't call)
getPreContent()
protected Virtual function that lets subclasses make content before the content.
getPostContent()
protected Virtual function that lets subclasses make content after the content.
_toID()
protected Converts a string input (usually the title) into an ID.
toHTML()
protected Processes plaintext (ripped from the source files) and converts them to HTML.
finalize()
protected Ran when building is done.
registerChild()
protected Registers a block as a child of this block.

Functions

unregister()
Removes the block from the tree.
parseTag()
Parses a tag (A delegate task of ScBlock()).
getGroup()
Returns the group where the block belongs to.
isHomePage()
Checks if the current block is the home page.

Tag functions

getValidTags()
Returns a list of valid tags in associative array format.
hasTags()
Checks if the current block has tags defined.
getTags()
Returns the tags for the block.

Data functions

Block type functions

getType()
Returns the typename.
getTypeName()
Returns the type name as shown in the first line.
getTypeData()

Traversion functions

getChildren()
Returns the list of children.
hasChildren()
Checks if the block has children.
getParent()
Returns the parent block, or NULL if none.
getAncestry()
Returns all the parent nodes.
hasParent()
Checks if the current block has a parent.
getMemberLists()
Returns a grouped list of member items under the block.
